## Features
Personal helper for interview preparation and access point for interviewers 
The goal is to Production-ready and deployable on the cloud  — not just a demo

## Tech Stack
- **Frontend:** Next.js.
- **Backend:**  FastAPI backend containerized with Docker and deployed with AWS Fargate
- **Database:** PostgreSQL for structured data and Qdrant (or pgvector) for vector embeddings
